# KBG-thon
# AI-Based Antibiotic Misuse Awareness & Risk Checker

A Streamlit-based web application developed for the KBG Club Recruitment Hackathon at IIT Mandi.

## 🚀 Features
- **Interactive Risk Checker**: A deterministic logic engine tracking antibiotic misuse habits.
- **AMR AI Assistant**: A conversational hub built to safely answer public health questions.
- **Myths vs. Facts**: Educational breakdown of antimicrobial misconceptions.

## 📚 Verified Data Sources
This project relies strictly on verified guidelines from national and global health organizations:
- **WHO**: [Antimicrobial Resistance Fact Sheet](data/amr-factsheet.pdf) (uploaded in repo)
- **ICMR**: [Validation of Rapid Diagnostics for AST Guidance (2025)](data/1736941384_guidancedocument_amrdiagnostics_revised.pdf) (uploaded in repo)
- **CDC**: [CDC Antibiotic Use Communication Resources](https://www.cdc.gov/antibiotic-use/communication-resources/index.html)

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **UI Framework**: Streamlit
